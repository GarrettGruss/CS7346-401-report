\section{Cloud Services Selection and Justification}

The system is built entirely on AWS, as this provider offers the most comprehensive suite of managed services for IOT device management. The table below summarizes each selected service and its role.

\begin{itemize}
  \item \textbf{Amazon CloudFront} --- Delivers static assets and cacheable content to consumers via AWS edge locations, reducing latency for the web and mobile frontend.
  \item \textbf{AWS WAF} --- Filters inbound requests for common web exploits and abusive traffic patterns before they reach backend services.
  \item \textbf{Amazon Cognito} --- Manages consumer identity, authentication flows, token issuance, and optional multi-factor authentication.
  \item \textbf{Amazon API Gateway} --- Provides the managed HTTPS entry point for all client requests, handling routing, throttling, and authorization enforcement.
  \item \textbf{AWS Lambda} --- Executes stateless control-plane logic --- command validation, event processing, and integration tasks --- in response to API and messaging events.
  \item \textbf{AWS Step Functions} --- Orchestrates multi-step automation workflows that require branching, waiting, retries, and explicit state tracking.
  \item \textbf{AWS IoT Core} --- Provides secure MQTT messaging, device identity, and policy enforcement for all non-video Device Controllers.
  \item \textbf{Amazon Kinesis Video Streams} --- Ingests, buffers, and retrieves camera feeds, providing a dedicated pipeline for video traffic separate from device telemetry.
  \item \textbf{Amazon DynamoDB} --- Stores current device state, automation definitions, household configuration, and command status with low-latency key-value access.
  \item \textbf{Amazon Timestream} --- Stores time-ordered sensor telemetry --- temperature history, motion events, power readings --- with retention policies suited to time-series data.
  \item \textbf{Amazon S3} --- Provides durable object storage for archived video clips, camera snapshots, and retained log files.
  \item \textbf{Amazon CloudWatch} --- Centralizes logs, metrics, alarms, and dashboards for operational visibility across all platform services.
  \item \textbf{Amazon SNS} --- Delivers consumer alert notifications --- push, SMS, and email --- and fans out internal platform events to downstream subscribers.
\end{itemize}
